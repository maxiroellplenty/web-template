# web-template
A quick template for a website using scss and bootstrap

## Install Packages:
* ``` npm install ```

## Build
*  run: ``` sh build.sh ```
* Edit paths at gulpfile.js

## Deploy
* run: ``` sh deploy.sh ```
* Update the branch & ssh connection at 
`deploy.sh` file.

## Create new view
* run: `sh cv.sh`
